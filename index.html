What You're Looking At — Ad Creative Explained
📱 These are Responsive Search Ads (RSAs) — Google's standard search ad format. Each ad contains up to 15 headlines and 4 descriptions. Google's AI mixes and matches combinations to find what performs best for each search query.
🎯 What you see below is a representative preview of how each ad appears in Google search results — showing the URL, 3 headlines (Google picks which 3 to show), and 2 descriptions. The actual ad shown to any individual will vary.
📊 This is the May 2026 ad inventory. There are 16 active search ads across Forklift, Dogging, Rigging and CSQ, plus display image ads for remarketing.
Google Ads Expert Observations
⚠ NSW Dogging Ad Is Weakest
The NSW Dogging ad opens with "Looking to get your dogging ticket? You are in the right place." — a generic opener used across multiple ads. The VIC version has location-specific, high-intent copy. This copy weakness may be contributing to NSW's zero-conversion result in May.
⚠ Generic HRWL Ad Across All States
One generic "High Risk Work Licence" RSA runs in NSW, QLD and VIC campaigns simultaneously. It has no state-specific headline. When someone in Melbourne searches "rigging course Melbourne," this ad shows no Melbourne-specific copy — reducing relevance and Quality Score.
⚠ "Customers Say..." Overused
"Customers Say 'Professional people, good trainers...'" appears as Description 3 in nearly every single ad. Google tests all description combinations — this repetition wastes a description slot and reduces variation Google can test. Replace half with specific proof points (5-star rating count, years operating, RTO number, job placement rates).
✅ VIC Dogging Melbourne Ad — Best in Account
The VIC Dogging Melbourne ad has the strongest copy of all ads: location-specific ("Carrum Downs", "Melbourne"), course code prominent (CPCCLDG3001), hands-on claim, WorkSafe reference. This directly mirrors how a tradesperson searches. Correlates with VIC Dogging's $67 CPA — the best Dogging result.
✅ CSQ Ad Well Structured
The CSQ ad correctly targets the funded-training search intent with headlines like "CSQ Funding Queensland", "Explore CSQ Funded Courses" and "Construction Skills Queensland." Landing page goes to /course-types/csq-courses/ which is correct. Clean and purpose-built.
⚡ Price in Rigging Intermediate Ad
The QLD Rigging Intermediate ad includes "Starts at $1686" as a headline. This is double-edged — it qualifies leads (only serious buyers proceed) but may reduce CTR. Worth A/B testing a version without the price to compare conversion quality vs volume.
⚡ "Reduced Prices" in VIC Dogging
One VIC Dogging variant includes "Reduced Prices" as a headline. This is a strong conversion lever but needs to be verified — if prices aren't actually reduced, this is misleading and risks policy violations. Confirm with the team whether this is accurate.
⚡ No Forklift NSW Evening Training Ad
Two NSW Forklift ads include "Evening Training Available" as a headline. However this is buried at headline position 11–12. If evening training is genuinely available and in demand, this should be a pinned Headline 1 or 2, not an optional rotation — it's a strong differentiator for working adults.

Step-by-Step: Clean Conversion Setup
1
Move Google Tag from footer to <head> on kallibr.com.au
The Google tag (AW-820318163) is currently placed in the WordPress footer — it fires too late and misses some page views. It must be in the <head>.
  1. Go to WordPress Admin → Avada → Theme Options → Advanced → Google Analytics
  2. Remove the tag from there if present
  3. Install Google Tag Manager via the official GTM WordPress plugin
  4. Add your GTM container code in GTM — this replaces the raw gtag.js and makes all future tracking changes easy (no more dev deployments for every tag change)
Dev · 30 mins
2
Track Gravity Forms Course Enquiry (Form 3) — already has redirect
Form 3 already redirects to /contact-thank-you/ after submission. You just need to fire a conversion tag on that page.
  1. In Google Ads → Goals → Conversions → + New Conversion Action
  2. Select Website → URL contains → contact-thank-you
  3. Name: Course Enquiry Submitted · Category: Submit lead form · Value: $75 · Count: One
  4. Google provides a tag snippet — add this to GTM as a new tag firing on pages matching /contact-thank-you/
  5. Verify in Tag Manager Preview mode: submit a test enquiry and confirm the tag fires
Dev + Agency · 20 mins
3
Track Gravity Forms Pre-Enrol (Form 10) — via JavaScript event
Form 10 doesn't redirect to a thank-you page. Instead, use Gravity Forms' built-in JavaScript event which fires on successful submission.
  1. In GTM, create a new Custom Event Trigger listening for: gform_confirmation_loaded
  2. Add a condition: Form ID equals 10 (so it only fires for this specific form, not all Gravity Forms)
  3. Create a Google Ads Conversion Tracking tag (Conversion ID: AW-820318163, Conversion Label: get from Google Ads) fired by that trigger
  4. Name the conversion: Pre-Enrolment Form Submitted · Value: $150 · Count: One
  5. Alternatively: add a redirect to a /enrolment-thank-you/ page in Gravity Forms settings → Form 10 → Confirmations → Redirect
Dev + Agency · 45 mins
4
Track HubSpot Pre-Enrolment Forms — add redirect to thank-you page
HubSpot forms currently show an inline thank-you message. Google's tag cannot detect this. The cleanest fix is adding a redirect after submission.
  1. In HubSpot → Marketing → Forms → Pre-Enrolment V2 (enrolling myself)
  2. Click Options → Redirect to another page
  3. Enter URL: https://kallibr.com.au/enrolment-confirmed/
  4. Create that page in WordPress (simple thank-you page — no content needed, Google tag will fire on it)
  5. Repeat for Pre-Enrolment V2 (enrolling others) and CSQ Apply Now
  6. In Google Ads: create conversion action → URL contains → enrolment-confirmed → Name: Enrolment Form Submitted · Value: $200 · Count: One
Dev + HubSpot Admin · 1 hour
5
Remove all junk conversion goals from Primary actions
Once the above conversions are live and recording, clean up the goals list.
  1. In Google Ads → Goals → Conversions → Summary
  2. For each of these goals, click Edit Goal → change from Primary to Secondary: Page View, Get Directions, Downloads, Sign-ups, Begin Checkout, Purchase, Store Visit, Other, Engagement, Contact (unless confirmed as form submissions)
  3. Keep as Primary: Phone Call Leads (60 sec), Course Enquiry Submitted, Pre-Enrolment Form Submitted, Enrolment Form Submitted
  4. Wait 2–3 weeks before touching Target CPA settings — Smart Bidding needs time to re-calibrate on the new clean signals
Agency · 15 mins
6
(Advanced) Capture GCLID for future offline conversion upload
For the ultimate tracking — matching Google Ad clicks to actual aXcelerate bookings — you need to capture the Google Click ID (GCLID) on form submission and store it in HubSpot.
  1. In HubSpot → Forms → Pre-Enrolment: add a hidden field named gclid
  2. In GTM: create a URL parameter variable reading the gclid value from the page URL
  3. On the enrol page load: populate the hidden HubSpot field with the GCLID value via GTM dataLayer push
  4. In HubSpot CRM: the GCLID is now stored on each contact record
  5. Weekly process: export confirmed bookings from aXcelerate → match to HubSpot contact via email → pull GCLID → upload CSV to Google Ads as offline conversions
  6. This gives Smart Bidding the true conversion signal: a confirmed course booking, not just a form submission
Dev + Agency · 3–4 hours setup + weekly 30 min process
